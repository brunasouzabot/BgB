Operações flexíveis

Lançamento rápido

Opere com tranquilidade

1. Acesse "Convidar lojas"

No painel 99Food Admin, clique no botão amarelo no canto direito.

2. Copie o código

Clique em "Gerar código" e copie o código de convite gerado.

3. Acesse o link de cadastro

Acesse o link cadastro.99food.com.br e inicie o processo de cadastro.

4. Complete o cadastro

Preencha os dados da loja, incluindo razão social, CNPJ, endereço e telefone.

5. Cole o código

No final do cadastro, cole o código de convite copiado anteriormente.

6. Envio para análise

Finalize o envio e aguarde a aprovação da equipe da 99Food.
